# 1. HGA : Article
### 기사 본문 크롤링 및 han.gl 링크로 자동 단축

기사 링크 입력시
- 링크 자동으로 han.gl 로 단축 (제공된 API 사용)
- 기사 제목, 언론사, 본문 내용(작성 날짜/시간 포함) 정보 크롤링

```
pyinstaller HGA.spec
```
